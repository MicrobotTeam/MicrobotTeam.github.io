### 怎么本地运行

```bash

# 进入目录
$ cd core

# 安装依赖
$ npm install

# 本地运行,localhost:9000
$ npm run dev

# 打包
$ npm run build

```

### 目录说明

```bash

    .
    ├── blog    # 旧版博文的存放目录
    ├── core    # 核心代码的存放目录
    ├── logo    # 公司logo的存放目录
    ├── static  # cd core 后npm run build 生成的资源存放目录
    ├── LICENSE # 许可证，这里采用了 Apache License 2.0
    ├── README.md   # 仓库的说明文档
    └── index.html  # cd core 后npm run build 生成的入口文件

```
